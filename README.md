# HPCS SS Spack Repository

## Use

```
git clone https://github.com/tsukuba-hpcs/spack-packages
spack repo add ./spack-packages
```

## Packages

- `chfs`
- `finchfs` ad hoc parallel file system for HPC
- `ior`
- `ndctl`
- `pmdk`
- `pmemkv`
- `libpmemobj-cpp`
- `rdbench`
- `rocksdb-spdk` rocksdb with spdk integration
- `libtraceevent`
- `libtracefs`
- `libudev`
